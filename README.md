## 🧠 Description

**PhantomReconX** is a powerful and lightweight Termux-based cybersecurity and reconnaissance tool designed for ethical hackers and security enthusiasts.
It enables efficient network scanning, information gathering, and automated reconnaissance directly from an Android environment.

Built with simplicity and performance in mind, PhantomReconX allows users to perform essential recon tasks quickly without the need for heavy systems.

---

## ⚙️ Features

* 🔍 Fast network reconnaissance
* 📡 Information gathering
* ⚡ Lightweight & optimized for Termux
* 🛠️ Easy to use interface
* 📁 Automatic report generation

---

## 🚀 Installation

```bash
pkg update && pkg upgrade -y
pkg install git python -y
git clone https://github.com/KushVerma-EthicalHax/PhantomReconX
cd PhantomReconX
python phantomreconx.py
```

---

## 📌 Usage

* Run the tool using the command above
* Enter the target (IP / domain) when prompted
* Results will be saved in the `reports` directory
